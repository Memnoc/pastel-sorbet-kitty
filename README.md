## Ingredients (The Palette)

**Pastel Sorbet** is the official terminal port of the [pastel-sorbet.nvim](https://github.com/Memnoc/pastel_sorbet.nvim) colorscheme. 
Rich, soft and pastel colors, featuring a mix of rich purples and vibrant accents:

| Flavor | Hex | Role |
| :--- | :--- | :--- |
| **Blackberry Base** | `#1c1826` | The deep, matte background. |
| **Taro Purple** | `#c88df7` | A rich primary accent for keywords. |
| **Strawberry Sorbet** | `#ff8596` | Soft reds for deletions and errors. |
| **Cool Mint** | `#6ef298` | Refreshing greens for strings. |
| **Blueberry** | `#7db8ff` | Calming blues for functions. |
| **Lemon Zest** | `#ffd85c` | Bright yellows for warnings. |
| **Whipped Cream** | `#e8e4f2` | High contrast foreground text. |

## Serving Suggestions (Installation)

### 1. Scoop the file
Save the configuration file to your Ghostty configuration directory.
Or, use your mouse and access the config directly from the Settings menu!

Ghostty uses a text config file, and normally is called "config" (with no quotes).

```bash
On Linux:
~/. config/kitty/kitty.conf if the XDG environment is not set. 
```
## Mix it in!

Place the file `kitty.conf` in your `./config/kitty/kitt.conf`.
You may (most definitely) need to restart Kitty.
